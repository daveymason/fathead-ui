Wrap your app in the root class and set theme:
```html

  <!-- Header, Navbar, Components... -->
</body>
```

- Alerts: Four severity levels with icons
- Navigation: Horizontal, vertical and tab styles
- Copy Buttons: Hologram feedback animation
