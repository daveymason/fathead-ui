Getting Started

Quick Setup

Add the stylesheet link below to your HTML and wrap your content in a fh-ui class.

Add to your project:

<link rel="stylesheet" href="https://daveymason.github.io/fathead-ui/fathead-ui.css" />

Simple Installation

Just add the stylesheet link to the head of your HTML document.

Golden Ratio & Spacing

Mathematical Harmony

Fathead UI uses the mathematical golden ratio (1.618) for all spacing and sizing to create naturally pleasing visual harmony.

CSS Variables

--fh-base: 1rem /* base unit */
--fh-xs: calc(var(--fh-base) / 1.618)
--fh-sm: calc(var(--fh-xs) * 1.618)
--fh-md: var(--fh-base)
--fh-lg: calc(var(--fh-base) * 1.618)
--fh-xl: calc(var(--fh-lg) * 1.618)
--fh-xxl: calc(var(--fh-xl) * 1.618)

Visual Representation

--fh-xl
--fh-lg
--fh-md
--fh-sm
--fh-xs

Pro Tip

Use these variables for margin, padding, gap, and font-size to create visually balanced and harmonious layouts that follow natural proportions.

HCI & Accessibility

Accessibility First

All Fathead UI components are designed with accessibility as a core principle, not an afterthought.

Visual Accessibility

  • High-contrast color schemes in both light and dark modes
  • WCAG 2.1 AA compliant text contrast ratios
  • Focus indicators for improved keyboard navigation
  • Consistent spacing with golden ratio proportions

Keyboard & Screen Reader

  • Full keyboard navigation support
  • Properly labeled form elements and controls
  • ARIA attributes for enhanced screen reader support
  • Semantic HTML structure throughout components

Motion & Animation Control

Reduced Motion Support
@media (prefers-reduced-motion) {
  .fh-ui * {
    animation-duration: 0.001ms !important;
    transition-duration: 0.001ms !important;
  }
}

Tailored to Your Brand

Fathead UI offers extensive customization via CSS variables (custom properties) that control everything from colors and spacing to component behavior.

Color Theme

:root {
  --fh-primary: #2DC0FF;
  --fh-accent: #FF8A00;
  --fh-secondary: #6A0DAD;

Component Styling

  --fh-glass-blur: 15px;

Theme Switching
Theme Attributes
<body class="fh-ui" data-theme="light">
<!-- Content -->
</body>

Toggle between "light" and "dark" themes with JavaScript:

document.body.setAttribute(
  'data-theme', 
  'dark'
);
Custom Themes

Create additional themes by adding CSS selectors:

.fh-ui[data-theme="neon"] {
  --fh-primary: #0ff;
  --fh-accent: #f0f;
  --fh-glass-bg: rgba(0,0,30,0.7);
}

Best Practice

Keep your custom CSS in a separate file and import it after the Fathead UI stylesheet to override default styles without modifying the original library.

Lightweight By Design

Code Examples

Example: CSS-First Modal

<!-- Hidden checkbox for state management -->
<input type="checkbox" id="my-popup-checkbox" class="fh-ui-popup-checkbox">

<!-- Trigger label styled as a button -->
<label for="my-popup-checkbox" class="fh-ui-button">Open Popup</label>

<!-- The modal/popup -->
<div class="fh-ui-popup" id="my-popup">
  <div class="fh-ui-popup-content">
    <p>Popup content</p>
    <label for="my-popup-checkbox" class="fh-ui-button">Close</label>
  </div>
  <label for="my-popup-checkbox" class="fh-ui-close-overlay"></label>
</div>
