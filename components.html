Buttons

Versatile button styles with hover effects, gradient background, and accessibility built-in. Use for primary and secondary actions.

<button class="fh-ui-button">Primary</button>
<button class="fh-ui-button" style="background: var(--fh-accent)">Accent</button>
<button class="fh-ui-button" style="background: var(--fh-secondary)">Secondary</button>
<button class="fh-ui-button" style="background:transparent;color:var(--fh-text-secondary);border:1px solid var(--fh-text-secondary)">Ghost</button>

Cards

Glassmorphic cards with hover effects for content grouping. Perfect for articles, product displays, or information grouping.

Card Title

Content with glassmorphism, microfeedbacks and a hover lift effect.

<div class="fh-ui-card">
  <h3>Card Title</h3>
  <p>Content with glassmorphism, microfeedbacks and a hover lift effect.</p>
</div>

Layout Grid

Responsive grid layout using CSS Grid. Automatically adjusts columns based on available width using the golden ratio.

Item 1
Item 2
Item 3
<div style="display:grid;grid-template-columns:repeat(auto-fill,minmax(200px,1fr));gap:var(--fh-lg)">
  <div class="fh-ui-card">Item 1</div>
  <div class="fh-ui-card">Item 2</div>
  <div class="fh-ui-card">Item 3</div>
</div>

Table

Clean and readable tables with subtle borders and consistent spacing. Great for data presentation.

Name Email Role

Bob bob@example.com User

    <tr>
      <th style="border-bottom:1px solid var(--fh-glass-border);padding:var(--fh-sm)">Name</th>
      <th style="border-bottom:1px solid var(--fh-glass-border);padding:var(--fh-sm)">Email</th>
      <th style="border-bottom:1px solid var(--fh-glass-border);padding:var(--fh-sm)">Role</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td style="padding:var(--fh-sm)">Alice</td>
      <td style="padding:var(--fh-sm)">alice@example.com</td>
      <td style="padding:var(--fh-sm)">Admin</td>
    </tr>
    <tr>
      <td style="padding:var(--fh-sm)">Bob</td>
      <td style="padding:var(--fh-sm)">bob@example.com</td>

    </tr>
  </tbody>
</table>

Forms

Styled form inputs with focus states and consistent sizing. Includes text fields, select dropdowns, and textareas.

<form>
  <input class="fh-ui-input" type="text" placeholder="Your Name" />
  <select class="fh-ui-select">
    <option>Option 1</option>
    <option>Option 2</option>
  </select>
  <textarea class="fh-ui-textarea" rows="3" placeholder="Your Message"></textarea>
  <button class="fh-ui-button">Submit</button>
</form>

Loader

CSS-only loading indicators for async operations. Includes spinners in various sizes and a full-screen overlay option.

<!-- CSS-only Spinner sizes -->
<div class="fh-ui-spinner sm"></div>
<div class="fh-ui-spinner"></div>
<div class="fh-ui-spinner lg"></div>
<div class="fh-ui-spinner xl"></div>

<!-- Loader Overlay with auto-hiding -->
<input type="checkbox" id="loader-overlay-checkbox" class="fh-ui-loader-checkbox">
<div id="fh-ui-loader-overlay">
  <div class="fh-ui-spinner xl"></div>
</div>
